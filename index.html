Settings

App, sync & account controls.
Your identity on this device
Every device/browser picks its own identity here — it's how the app knows whose permissions to apply, and it's used nowhere else. Not a login, just a local "who's using this device right now" setting.
Not listed yet? Create a new Owner or Staff identity right here — this adds them to the Vendors & Staff list too.
Appearance & language
Applies to this device only, after you click Apply. Translation currently covers navigation, page titles, the Dashboard, and key buttons — most other text (tables, forms) stays in English for now. A QR color that's too light to scan reliably is automatically ignored in favor of black.
View prices in another currency
Converts how prices are DISPLAYED on this device only — never changes what's actually recorded. Uses free daily exchange rates from exchangerate-api.com.
Permissions UI-level only — not real security
Owners always have full access and aren't shown here — that's fixed, not editable, with a couple of columns marked ⏳ reserved for finer-grained control later. This only hides/disables buttons for convenience; it can't stop someone who opens dev tools, since there's no server or login behind it.
PIN / Pattern sign-in deters casual misuse — not real security either
Setting a PIN or pattern for someone means the app will ask for it on load and use it to identify who's on the device, instead of picking from a list. Both are stored as a simple scrambled value, not real encryption — anyone reading this file's own code could work a short one out. There's no password recovery: if it's forgotten, an Owner resets it here. 5 wrong attempts blocks the account until an Owner unblocks it.
GitHub sync — single source of truth
Every add/edit here pushes Entries, Products, Bills, Staff and Vendors as JSON files to a folder in a GitHub repo you choose — readable from any device, whether this page is opened inside Claude or downloaded and run on its own. Use a fine-grained personal access token scoped to only this one repo with Contents: Read and write. The token stays on this device only; it is never written into the synced files.
Backup & export
Separate CSV files, one per record type — stand-ins for separate Excel sheets — plus a full JSON backup you can restore later.
⚠ Danger zone Owner only
Reset this device's view
Clears what's shown here and pauses GitHub sync. Your GitHub repo is untouched — flip sync back on later to bring it all back.
Erase everything — no exceptions
Deletes all shop data locally AND overwrites every file in the connected GitHub repo with nothing. Cannot be undone.
What this app still can't do
It can't send WhatsApp or email automatically — that needs a connected messaging service. It can't read a price off a photo — camera scan only attaches a picture. OneDrive/Google Drive sync isn't wired up — that needs an OAuth app registered on your end first; GitHub sync above is the working single-source-of-truth for now.
